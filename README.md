The App 
## Usage
To run the project from commandline:
```
git clone https://github.com/GavintheAmateur/travel-planner
cd travel-planner
npm i
npm start
```
Then you will be able to play with the App at: [http://localhost:8081/](http://localhost:8081/)
## Technology
